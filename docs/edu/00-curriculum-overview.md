## 교육 개요

| 항목 | 내용 |
|------|------|
| 교육명 | 클로드 코드를 활용한 민생위기 조기경보 웹앱 개발 |
| 대상 | 비전공자 3명 |

## 회차별 상세

### 1차: 환경설정 + 클로드 코드 & MCP 체험

**이론 (60분)**
- 웹앱이란 무엇인가 (브라우저에서 동작하는 앱)
- 클로드 코드 소개 (AI가 코드를 대신 작성해주는 도구)
- MCP 서버란? (클로드 코드에 추가되는 전문 도구)
  - shadcn: UI 컴포넌트 카탈로그
  - sequential-thinking: 복잡한 문제 단계별 사고
  - context7: 최신 기술 문서 실시간 검색
  - playwright: 브라우저 자동 테스트
- 프롬프트 기본 원칙 (명확하게, 구체적으로, 단계별로)

**실습 (120분)**
- 개발 환경 설치 (Node.js, VS Code, Claude Code CLI)
- 클로드 코드로 첫 프로젝트 생성 + 자기소개 페이지 만들기
- MCP 서버 4개 실습 체험
  - shadcn: 컴포넌트 검색 + 설치
  - sequential-thinking: 단계별 설계 과정 관찰
  - context7: 기술 문서 조회
  - playwright: 스크린샷 + 화면 자동 확인

**산출물**: 개인별 자기소개 웹페이지 + MCP 4개 체험 완료

---

### 2차: UI 구현 - 프롬프트로 화면 만들기

**이론 (60분)**
- 컴포넌트 개념 (레고 블록처럼 조립하는 UI 조각)
- 이르미 프로젝트 3개 탭 구조 소개 (대시보드/위기신호/뉴스분석)
- 데이터가 화면에 표시되는 흐름 (JSON -> 컴포넌트 -> 화면)
- UI 프롬프트 작성법 (레이아웃 -> 컴포넌트 -> 스타일 순서)

**실습 (120분)**
- 이르미 프로젝트 클론 + 실행
- 브라우저에서 현재 대시보드 살펴보기
- 클로드 코드로 간단한 카드 컴포넌트 만들기
- 기존 컴포넌트 수정 프롬프트 실습 (텍스트/레이아웃 변경)
- 필터 기능 동작 확인 + 수정 프롬프트
- shadcn 컴포넌트 활용 프롬프트

**산출물**: 이르미 프로젝트에 본인이 수정한 UI 1개 이상

---

### 3차: AI API 활용

**이론 (60분)**
- API란? (음식점 주문서처럼 요청하면 응답이 오는 구조)
- Claude API 기본 개념 (system prompt, user message, 응답)
- 환경변수(.env.local)와 API 키 관리
- API Route 개념 (클라이언트 -> 서버 -> 외부 API)

**실습 (120분)**
- .env.local 파일 설정 실습
- 클로드 코드로 간단한 AI 채팅 기능 테스트
- system prompt 수정으로 AI 응답 스타일 변경
- 뉴스 분석 API 호출 테스트
- 보조금24 API / 국회 API 데이터 확인
- 프롬프트 튜닝: 더 나은 분석 결과 만들기

**산출물**: AI 응답이 동작하는 기능 1개 테스트 완료

---

### 4차: 해커톤 데이터 적용

**이론 (45분)**
- 현재 목업 데이터 vs 실제 데이터 차이점
- JSON 데이터 구조 설명 (필드별 의미)
- 데이터 교체 절차 (파일 교체 -> 매핑 확인 -> 테스트)
- 해커톤 당일 체크리스트

**실습 (135분)**
- data/mock/ 폴더의 JSON 파일 구조 분석
- 새로운 JSON 데이터 파일 교체 프롬프트 실습
- 데이터 필드가 다를 때 매핑 수정 프롬프트
- API Route 목업 -> 실제 전환 프롬프트
- 전체 3개 탭 동작 테스트 (브라우저 확인)
- 에러 발생 시 클로드 코드로 디버깅 실습

**산출물**: 새 데이터가 적용된 동작하는 웹앱

---

### 5차: 복기 + 실전 연습

**이론 (60분)**
- 전체 교육 내용 5분 퀴즈 (가벼운 복습)
- 효과적인 프롬프트 패턴 총정리
- 디버깅 기본: 에러 메시지 읽기 + 프롬프트로 해결
- 해커톤 당일 전략 수립 + 역할 분담 논의

**실습 (120분)**
- 종합 실습: 새로운 기능 하나를 처음부터 끝까지 구현 (60분)
  - 예시: 새 대시보드 위젯 추가, 필터 기능 개선 등
- 미니 해커톤 시뮬레이션 (30분)
  - 실제 해커톤처럼 시간 제한 + 목표 설정
- 트러블슈팅 실습 (15분)
  - 의도적으로 만든 에러를 프롬프트로 해결
- Q&A + 해커톤 최종 준비 (15분)

**산출물**: 종합 실습 결과물 + 해커톤 준비 완료
